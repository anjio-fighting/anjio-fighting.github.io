Hexo-GitHub搭建个人博客

一、Hexo搭建步骤

1、安装git

在git官网上下载对应电脑版本的文件
，安装选项全部默认，当添加路径时选择——Use Git from the Windows Command Prompt

2、安装node.js

在官网上下载即可，安装选项默认，一路点击Next

3、Win+R，打开cmd，输入node -v和npm -v，若出现版本号则安装成功

4、安装Hexo

创建文件夹Blog
打开git bash（以管理员身份进入)
然后cd到Blog文件夹下
输入npm install -g hexo-cli ,安装Hexo
输入hexo -v检验安装是否成功
再输入hexo init初始化文件
新建完成后，指定文件夹Blog目录有一些文件

  • node_modules:依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：存放文章
  • themes：主题
  • _confid.yml：博客配置文件

输入hexo g生成静态网页
再输入hexo s打开本地服务器，Ctrl+C关闭本地服务器

二、注册GitHub账号并创建个人仓库

（ 一 ）GitHub的账户注册

1、输入地址：https://github.com/
2、点击右上角的注册按钮sigh up
3、输入自己的邮箱、创建密码、创建用户名
4、进行相应的验证，之后填入QQ邮箱收到的验证码 5、根据自身情况进行个性化设置（可跳过）

（ 二 ）GitHub搭建个人博客

1、登录账号后，点击右上角的“+”按钮，选择“New repository”

2、在“Repository name”输入仓库名称 （名称格式:你的GitHub仓库名.github.io——这是以后访问的域名）![](C:/Users/14014/Pictures/Screenshots/屏幕截图 2023-10-28 124201.png) 3、选择public

4、点击“Create respository”
5、仓库创建好后，把仓库地址复制下来留到后面配置时使用（使用ssh）

type: git
repository: git@github.com:用户名/用户名.github.io.git
branch: main
